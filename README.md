# FIFA World Cup 2026 Player Performance Analytics

# Project Overview
This project analyzes the FIFA World Cup 2026 Player Performance Dataset using PostgreSQL, SQL, Excel, and Power BI. The objective is to transform raw football data into meaningful insights through data cleaning, SQL analysis, and interactive dashboards.
The dataset contains over 54,600 player performance records from the FIFA World Cup 2026, including player information, match statistics, passing metrics, attacking performance, defensive actions, and player ratings.

# Objectives

- Import and manage a large dataset using PostgreSQL
- Perform data analysis using SQL
- Answer business questions with SQL queries
- Create interactive Power BI dashboards
- Demonstrate an end-to-end data analytics workflow
